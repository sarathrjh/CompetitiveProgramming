1.There are 3 Application : Client Application, Student Course Web service, Data web Service
2.Database used is h2.
3. Steps to run the application
	1. Import 3 spring boot projects into eclipse
	2. Build the project with mvn clean install
	3. Set 9001 port for Client Application,8080 for Student Course Web service and 8082 for Data Service
	   in application.properties of each project.
	4. Run each application by run as Java application from main class of each project.
